# Devcontainer Environments

### Prerequsites
- Docker
- VS Code
- WSL debian-based distro if on Windows

### Linux/Mac

- Open project root in IDE
- `Ctrl + Shift + P` and select `Dev Containers: Open Folder in Container`
- Select option 
  - `local` for development on your machine
  - `server` if you're on a special server (ask rossy)

### Windows


- Make sure you have access to docker from your WSL distro `docker --version`
- Follow guide above for Linux/Mac
